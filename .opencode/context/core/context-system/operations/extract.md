# Extract Operation

Core concept: Extract converts raw sources (docs, code, URLs) into compact context artifacts aligned to project standards.

## Key Points
- Extract only stable knowledge that will be reused.
- Normalize terminology to project naming.
- Save into function-based folders by content type.
- Keep each artifact concise and linked to source.

## Required Reads
- `../standards/mvi.md`
- `../guides/compact.md`
- `../guides/workflows.md`

## Steps
```text
1) Read source
2) Identify reusable decisions and patterns
3) Draft MVI output
4) Place file in concepts/examples/guides/lookup/errors
5) Add source reference link
```
