# Error Operation

Core concept: Error operation captures recurring failures as reusable troubleshooting entries with clear fixes and prevention notes.

## Key Points
- Capture symptom, cause, fix, and prevention.
- Keep entries specific to real recurring issues.
- Link to source logs or docs when possible.
- Store in `errors/` under the target category.

## Required Reads
- `../standards/templates.md`
- `../guides/workflows.md`

## Steps
```text
1) Name the error clearly
2) Capture root cause
3) Add tested fix
4) Add prevention guardrail
5) Link reference
```
