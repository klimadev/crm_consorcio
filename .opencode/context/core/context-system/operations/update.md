# Update Operation

Core concept: Update refreshes context when tools, APIs, or frameworks change, while preserving local conventions and structure.

## Key Points
- Update impacted files only.
- Keep prior decisions unless explicitly superseded.
- Record changed behavior and migration notes.
- Re-validate examples after updates.

## Required Reads
- `../guides/workflows.md`
- `../standards/mvi.md`

## Steps
```text
1) Identify changed surface area
2) Edit affected context files
3) Add concise migration guidance
4) Update references and navigation
5) Validate scannability and line count
```
