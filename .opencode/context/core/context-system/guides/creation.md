# Creation Guide

Core concept: Create new context categories with function-based folders and compact starter files so contributors can navigate instantly.

## Key Points
- Use required structure: `concepts/`, `examples/`, `guides/`, `lookup/`, `errors/`.
- Add `navigation.md` first.
- Seed each folder with one minimal file if possible.
- Keep each file under 200 lines.

## Minimal Steps
```text
1) Create category directory
2) Create required function folders
3) Add navigation.md
4) Add minimal starter docs
5) Validate MVI + links
```

## Starter Checklist
- Folder structure complete
- navigation links work
- files are concise
