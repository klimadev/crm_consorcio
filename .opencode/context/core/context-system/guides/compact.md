# Compact Guide

Core concept: Convert verbose context into minimal, task-ready information without losing essential decisions.

## Key Points
- Preserve decisions, constraints, and examples.
- Remove narrative, duplicates, and old status notes.
- Keep one file focused on one topic.
- Enforce under-200-line output.

## Compact Procedure
```text
1) Keep core concept (1-3 sentences)
2) Keep 3-5 key points
3) Keep one minimal example
4) Add one reference link
```

## Good Output Signals
- Fast to scan
- Immediately actionable
- Easy to update
